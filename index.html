How Singlish Typing Works

Type using English letters — they're converted to Sinhala Unicode automatically as you type. The tables below show every key combination the engine understands, organized the same way as the official scheme reference.

ස්වර අක්ෂර (Vowels)

Write phonetically. For a long vowel, double the same English letter. Only ඇ and ඍ use capital letters.

ව්‍යංජන අක්ෂර (Consonants)

Write phonetically. Retroflex (මූර්ධජ) letters use a capital first letter. ‘ද’ can also be typed as q.

මහප්‍රාණ අක්ෂර (Aspirated letters)

Add h to the base consonant. Only ‘ඨ’ and ‘ඪ’ use a capital letter.

සඤ්ඤක සහ අනෙකුත් අක්ෂර (Prenasalized & other letters)

Add z before the base consonant. Only ‘ඹ’ uses a capital B.

පිළි සමඟ උදාහරණ (Letter + vowel-sign examples, using ‘ක’)

විශේෂ අක්ෂර (Special characters)

බැඳි අක්ෂර — Conjuncts (Yansaya, Rakaransha, Repaya)

Yansaya (◌්‍ය) forms automatically — type a consonant, then y, then a vowel: vidya → විද්‍යා. Rakaransha (◌්‍ර) works the same way with r: kra → ක්‍ර. Repaya (ර්‍◌) comes before the next consonant — type \r then the consonant. If a particular conjunct won't type correctly from Singlish, open the on-screen Keyboard panel (⌨️ button above the text box) — it has dedicated ◌්‍ය, ◌්‍ර, ර්‍◌ and ZWJ keys you can tap right after a base consonant to join them by hand.

Examples

How suggestions are generated

  1. As you type a word in Singlish (e.g. shakthiya), the engine converts it phonetically to Sinhala Unicode in real time — that's what appears in the text box and at the top of the popup.
  2. The popup then checks your exact typed word against a 145,000-entry Singlish → Sinhala dictionary for verified real-word spellings — these are usually more accurate than the phonetic guess alone.
  3. It also looks for longer dictionary words that start with what you've typed so far, so a short prefix like patam can predict a full word like පාඨමාලාව.
  4. If the dictionary has too few matches, frequency-ranked predictions from a 30,000-word corpus fill in the rest.
  5. Your raw typed text is always offered as the last numbered option, so you can keep it exactly as typed (useful for names or English words).

Using the suggestion popup

  1. As you type, a popup appears above (or below, near a screen edge) your cursor showing up to 8 suggestions.
  2. The top suggestion is the one currently shown inline in the text box.
  3. Press 1–9 on your keyboard to instantly pick a specific suggestion.
  4. Press Space or just keep typing to accept what's currently shown and move on.
  5. Use ↑ / ↓ to highlight a different suggestion, then Enter to accept it.
  6. Click any suggestion with your mouse or finger to select it.

If the word you want isn't suggested

  1. Try a different spelling variant — e.g. kohomada vs kohomadha can give different results, since single vs. doubled/aspirated letters map to different Sinhala consonants (see the scheme table above).
  2. Keep typing further into the word — longer prefixes narrow the dictionary match considerably.
  3. For rare words or proper nouns, the phonetic conversion is usually close — just edit it by hand afterwards.
  4. Try voice typing for words that are awkward to spell out phonetically.
